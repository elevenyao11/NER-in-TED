Task: Is the label(${label}) correct for the entity(${entities}) in this context?

...${before} ${entities} ${after}...
Label: ${label}

Decide whether the label is correct or not for the given entity.
Each entity is in bold, and it is around a short context.
If it is not correct, please choose the entity that you can think of the best fit.

For example,

... Since I was younger, I have been using iPhone because I love the design ...

Label: PRODUCT

The correct answer is YES, since iPhone is a product.

Your response will be used to correct errors of automatically-generated named entity labels developed as a part of an academic research project in parallel NER corpus construction.
